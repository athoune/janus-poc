Janus POC
=========

Doing stuff with webrtc, serverside and clientside.

Audio room for all.

Dev time
--------

Build the Janus image

    make janus-image

Start the containers

    make dev

Steal the js libraries

    make steal-janus-js-lib

Open the main page : http://localhost
